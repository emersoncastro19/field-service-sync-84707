# Manuales del Sistema

Esta carpeta contiene los manuales en formato PDF que se pueden descargar desde la aplicación.

## Archivos requeridos:

1. **manual-usuario.pdf** - Manual de Usuario (para todos los usuarios)
2. **manual-analista.pdf** - Manual de Analista (para agentes, coordinadores, técnicos y admins)

## Instrucciones:

1. Coloca los archivos PDF en esta carpeta
2. Asegúrate de que los nombres coincidan exactamente con los especificados
3. Los archivos se descargarán automáticamente cuando los usuarios hagan clic en los botones

## Ejemplo de estructura:
```
public/manuales/
├── manual-usuario.pdf
├── manual-analista.pdf
└── README.md
```
